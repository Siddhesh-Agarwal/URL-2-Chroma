> creates a `chroma_db` folder in the same path

## Setup

1. Clone the folder

   ```sh
   git clone https://github.com/Siddhesh-Agarwal/URL-2-Chroma.git
   ```

2. Install the dependencies

   ```sh
   poetry install
   ```

3. Begin the server

   ```sh
   streamlit run ./st_app.py
   ```
